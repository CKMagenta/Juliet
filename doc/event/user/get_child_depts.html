EVENT/USER:GET_CHILD_DEPTS

유저 등록

  • request payload
    • event / USER:GET_CHILD_DEPTS
    • data
      • dept_hash / 부서해시
  • response payload
    • 성공 시
      • event / USER:GET_CHILD_DEPTS
      • status / 1 (SUCCESS)
      • data ( 건제 순(DB 내부의 dept_code 순서)으로 정렬된 2dim array )
        • dept_hash / 부서 해쉬
        • dept_code / 부서 코드 (건제순 정렬용)
        • dept_name / 부서 이름
        • dept_full_name / 부서 전체 이름
    • 인수가 부족할 때
      • event / USER:GET_CHILD_DEPTS
      • status / 2 (INSUFFICIENT_ARGUMENTS)
      • data / "insufficient informations"
