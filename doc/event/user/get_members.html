EVENT/USER:GET_MEMBERS

해당 부서 소속 멤버 정보 가져오기

  • request payload
    • event / USER:GET_MEMBERS
    • data
      • dept_hash / 부서해시
      • fetch_all / 해당 부서의 하위 부서에 속한 사람들까지 모두 가져오는지 여부
  • response payload
    • 성공 시
      • event / USER:GET_MEMBERS
      • status / 1 (SUCCESS)
      • data ( 계급 순(DB 내부의 user_rank 순서)으로 정렬된 2dim array )
        • user_hash / 유저 해쉬
        • user_name / 유저이름
        • user_rank / 계급
        • user_duty / 직위
        • pic_filename / 프로필사진 파일 경로와 이름
    • 인수가 부족할 때
      • event / USER:GET_MEMBERS
      • status / 2 (INSUFFICIENT_ARGUMENTS)
      • data / "insufficient informations"
