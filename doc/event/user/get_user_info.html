EVENT/USER:GET_USER_INFO

사용자 정보 가져오기

  • request payload
    • event / USER:GET_USER_INFO
    • data
      • user_hash / 유저 해시 (string 또는 string array)
  • response payload
    • 성공 시
      • event / USER:GET_USER_INFO
      • status / 1 (SUCCESS)
      • data (input에 따라 1dim array 또는 2dim array)
        • user_hash / 유저 해쉬
        • user_name / 사용자 이름
        • dept_code / 부서 코드
        • dept_name / 부서 이름
        • dept_full_name / 전체 부서 경로 (스페이스로 구분)
        • user_rank / 계급
        • user_duty / 직위
        • pic_filename / 프로필사진경로
        • is_enabled / 활성화여부
    • 인수가 부족할 때
      • event / USER:GET_USER_INFO
      • status / 2 (INSUFFICIENT_ARGUMENTS)
      • data / "insufficient informations"
    • 등록되지 않은 유저일 때
      • event / USER:GET_USER_INFO
      • status / 3 (NO_DATA)
      • data / "unregistered user"
