EVENT/USER:REGISTER

유저 등록

  • request payload
    • event / USER:REGISTER
    • data 
      [ { "user_name":유저이름,
	"dept_hash": 부서 해쉬,
	"user_role": 직위
	"user_rank": 계급 } ]
  • response payload
    • 성공 시
      • event / USER:REGISTER
      • status / 1 (SUCCESS)
      • data 
        [ { "dept_hash": 부서 해쉬, 
	"user_hash": 유저 해쉬 } ]
    • 인수가 부족할 때
      • event / USER:REGISTER
      • status / 2 (INSUFFICIENT_ARGUMENTS)
      • data 
        [ { "msg":에러 메세지} ]
    • 등록되지 않은 부서일 때
      • event / USER:REGISTER
      • status / 3 (NO_DATA)
      • data 
        [ { "msg":에러 메세지} ]
