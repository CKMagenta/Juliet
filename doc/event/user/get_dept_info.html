EVENT/USER:GET_DEPT_INFO

부서 정보 가져오기

  • request payload
    • event / USER:GET_DEPT_INFO
    • data
      • dept_hash / 부서 seq 해시 (string 또는 string array)
  • response payload
    • 성공 시
      • event / USER:GET_DEPT_INFO
      • status / 1 (SUCCESS)
      • data (input에 따라 1dim array 또는 2dim array)
        • dept_hash / 부서 해쉬
        • dept_name / 부서 이름
        • dept_code / 부서 코드
        • dept_full_name / 전체 부서 이름
        • is_enabled / 활성화여부
        • parent_hash / 상위부서 해시
    • 인수가 부족할 때
      • event / USER:GET_DEPT_INFO
      • status / 2 (INSUFFICIENT_ARGUMENTS)
      • data / "insufficient informations"
    • 등록되지 않은 부서일 때
      • event / USER:GET_DEPT_INFO
      • status / 3 (NO_DATA)
      • data / "unregistered department"
