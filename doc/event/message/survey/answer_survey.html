EVENT/MESSAGE:SURVEY:ANSWER_SURVEY

설문조사에 응답

  • request payload
    • event / MESSAGE:SURVEY:ANSWER_SURVEY
    • data
      • user_hash / 유저 해시
      • survey_hash / 설문조사 seq 해시
      • answersheet / 설문조사 응답 정보를 담고 있는 json object
  • response payload
    • 성공 시
      • event / MESSAGE:SURVEY:ANSWER_SURVEY
      • status / 1 (SUCCESS)
      • data / "success"
