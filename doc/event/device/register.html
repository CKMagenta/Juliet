EVENT/DEVICE:REGISTER

사용자의 기기를 등록.

  • request payload
    • event / DEVICE:REGISTER
    • data 
      [ { "user_hash": 유저 해쉬,
	"uuid": 기기식별용 uuid,
	"regid": regid,
	"dev_type": device type (안드로이드 a, 아이폰 i, 피씨 p)
	} ]
  • response payload
    • 성공 시
      • event / DEVICE:REGISTER
      • status / 1 (SUCCESS)
      • data 
        [ { "dev_hash": 디바이스 seq 해쉬,
	"regid": gcm registration key,
	"uuid": 기기 식별용 uuid
 } ]
    • 실패 시
      • event / DEVICE:REGISTER
      • status / 2 (INSUFFICIENT_ARGUMENTS)
      • data 
        [{"msg":메세지}]
