EVENT/DEVICE:UNREGISTER

사용자의 기기를 등록해지.

  • request payload
    • event / DEVICE:UNREGISTER
    • data 
      [ { "uuid":기기 식별용 uuid,
	"regid": regid }]
  • response payload
    • 성공 시
      • event / DEVICE:UNREGISTER
      • status / 1 (SUCCESS)
      • data 
        [{"msg":메세지}]
    • 인수가 부족할 때
      • event / DEVICE:UNREGISTER
      • status / 2 (INSUFFICIENT_ARGUMENTS)
      • data 
        [{"msg":메세지}]
    • 삭제하려는 데이터가 DB에 없을 때
      • event / DEVICE:UNREGISTER
      • status / 3 (NO_DATA)
      • data 
        [{"msg":메세지}]
