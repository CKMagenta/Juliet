EVENT/DEVICE:UNREGISTER

사용자의 기기를 등록해지.

  • request payload
    • event / DEVICE:UNREGISTER
    • data
      • uuid / 기기식별용 uuid
      • regid / gcm registartion key
  • response payload
    • 성공 시
      • event / DEVICE:UNREGISTER
      • status / 1 (SUCCESS)
      • data / "success"
    • 인수가 부족할 때
      • event / DEVICE:UNREGISTER
      • status / 2 (INSUFFICIENT_ARGUMENTS)
      • data / "insufficient informations"
    • 삭제하려는 데이터가 DB에 없을 때
      • event / DEVICE:UNREGISTER
      • status / 3 (NO_DATA)
      • data / "no data in table"
