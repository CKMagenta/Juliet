EVENT/DEVICE:IS_REGISTERED

사용자의 기기가 등록되어 있는 지 확인.

  • request payload
    • event / DEVICE:IS_REGISTERED
    • data 
      [ { "user_hash": 유저 해쉬,
	"uuid": 기기 식별용 uuid,
	"regid": gcm registration key
} ]
  • response payload
    • 성공 시
      • event / DEVICE:IS_REGISTERED
      • status / 1 (SUCCESS)
      • data 
        [ { "isRegistered": 기기 등록 여부,
	"isEnabled": 기기 활성화 여부} ]
    • 실패 시
      • event / DEVICE:IS_REGISTERED
      • status / 2 (INSUFFICIENT_ARGUMENTS)
      • data 
        [{"msg": 메세지}]
