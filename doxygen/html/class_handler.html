Detailed Description

Payload Handler 클라이언트에서 http get 또는 post 요청으로 payload object를 json의 형태로 전송하면
이 컨트롤러의 call 모듈로 받아서 requestPayload['event'] 에 설정된 event type을 가지고
적절한 모델을 호출하여 요청을 처리한다.

그리고 각각의 모델에서는 이 객체의 static 변수인 responsePayload에
결과를 저장하고 마지막에 handler는 responsePayload를 json으로 변환하여 응답한다.
